Текст самого тестового задания:
Есть массив чисел:
[5,8,2,1,15,2,3,5,9,11,10,4,3,14,1,7,10,3,2,13]
Вывести его в виде графика (bar charts), где по оси x - индекс элемента, а по 
оси y - значение. Сделать цветовую градацию показателей на графике:0-5 зеленый, 6-10 жёлтый, 
свыше 10 красный. Допускается использование только html, css, javascript. 
Использование сторонних библиотек, canvas, svg не допускается.
Реализация:
Была реализована функция barChartBuilder(massife,height,width),
принимающая 3 аргумента:сам массив, высоту деления по оси y и ширину деления по оси x.
